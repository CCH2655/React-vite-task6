# React + Vite 專案

## 任務說明  
目標以完成購物網站頁面，以react router完成路由連接。  
- 可以展現產品、產品詳細、購物車操作

## 規格驅動開發SDD(Specification-Driven Development)
1. 建立頁面  
   - 首頁
   - 產品頁
   - 產品詳細頁
   - 購物車頁
2. 串接商品列表API
3. 確認並選擇單一產品API (useLoader or useParams)
4. 購物車API，添加購物車、更新購物車、刪除單一購物車、刪除全部購物車

## 技術
- React
- Vite
- Axios
- Bootstrap 5
- JavaScript (ES6)
