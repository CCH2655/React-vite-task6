# React RESTful API 串接

## 任務說明  
目標以串接 API 完成一頁式的產品新增、刪除、修改的頁面。  
- 可以新增、編輯、刪除商品  
- 商品啟用、關閉可以使用不同的顏色標示  

## 規格驅動開發SDD(Specification-Driven Development)
1. 製作所有版型
2. 串接登入功能
3. 轉到產品列表頁面
4. 取得所有產品列表的 API 串接
5. 彈出 Modal
6. 彈出 Modal 並取得單一產品資料
7. 串接更新資料 API
8. 關閉 Modal
9. 刪除功能製作...

## 技術
- React
- Vite
- Axios
- Bootstrap 5
- JavaScript (ES6)
