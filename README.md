# React + Vite 專案

## 任務說明  
目標以串接 API 完成一頁式的產品新增、刪除、修改的頁面。  
- 可以新增、編輯、刪除商品  
- 商品啟用、關閉可以使用不同的顏色標示  

## 規格驅動開發SDD(Specification-Driven Development)
1. 將後台頁面 Modal 以及分頁改成元件
2. 使用 import module 來引入元件
3. 使用者可以打開 Modal 新增、編輯、刪除商品
4. 串接圖片上傳 API 功能以及新增一個自訂欄位

## 技術
- React
- Vite
- Axios
- Bootstrap 5
- JavaScript (ES6)
